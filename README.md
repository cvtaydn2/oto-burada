# Free Car Marketplace MVP

A mobile-first, trust-oriented car-only classifieds platform where users can post listings for free.

## Stack
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- React Hook Form
- Zod
- TanStack Query
- Supabase Auth
- Supabase Postgres
- Supabase Storage
